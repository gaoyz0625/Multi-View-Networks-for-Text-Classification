This code is my implementation of the paper:
"Multi View network for text classification"
-----------------------------------------------------------
Software: python 3.5 ¶

downloaded and put in the same folder where the ipython file is
located.

-HP Computer

-RAM: 8Gig, Intel® HD Graphics 520 (Skylake GT2)
